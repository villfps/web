Portfolio

Computer Systems Servicing

Gain hands-on experience in assembling, maintaining, and troubleshooting computer hardware and networks, learning essential skills to ensure smooth operations. This subject delves into the intricate workings of computer systems, covering hardware, software, and networking. Students learn how to assemble, maintain, troubleshoot, and upgrade computer systems. They gain insights into the components that make up a computer, such as processors, memory, storage devices, and peripherals. Understanding computer system services is crucial in today's technology-driven world, as it provides the foundation for a wide range of IT careers, including system administration, technical support, and network engineering.

Web Development

Master the fundamentals of web design and coding, acquiring the ability to create visually appealing and interactive websites using HTML, CSS, and JavaScript. In the digital age, web development is a cornerstone of technological innovation. This subject empowers students to create dynamic and interactive websites and web applications. They learn programming languages like HTML, CSS, and JavaScript, along with frameworks and tools that streamline development processes. Students explore user interface design, responsive web design, database integration, and server-side scripting. With the skills gained in web development, students can pursue careers as web developers, front-end developers, full-stack developers, or even start their own web development ventures. Check out some of the websites I have made so far, Services and Menu.

Computer Programming

Develop problem-solving skills and learn programming languages like Python, Java, and C++, enabling you to create software solutions and algorithms to address real-world challenges. Computer programming is the art of writing instructions for computers to execute tasks. This subject introduces students to programming paradigms, data structures, algorithms, and problem-solving techniques. They learn programming languages such as Python, Java, C++, and more, gaining proficiency in writing code to solve real-world problems. Through hands-on projects and exercises, students enhance their logical thinking, creativity, and attention to detail. Proficiency in computer programming opens up diverse career opportunities in software development, game development, artificial intelligence, data science, and beyond.
